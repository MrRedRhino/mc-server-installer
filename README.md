### MC-Server-Installer

This tool downloads server-jars, creates configs and start-scripts.

### [DOWNLOAD](https://github.com/MrRedRhino/mc-server-installer/releases/)

### Supported server software:
 - Velocity-proxy
 - Paper
 - Vanilla
 - Purpur
 - Fabric
